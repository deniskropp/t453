# GEMINI.md - Recursive Consent-Gated Multi-Agent Co-Creative Ecosystem

## Project Overview
This project constitutes a "Recursive Consent-Gated Multi-Agent Co-Creative Ecosystem." It is a living Meta-Playbook designed to explore and implement recursive orchestration, automated consent gating, and multi-agent coordination.

## Notational Conventions
The project utilizes a `⫻` (KickLang) prefix notation for functional modules, data, and logic.
- `⫻context/`: System modules, ecosystem definitions, and structure maps.
- `⫻data/`: Artifacts produced and consumed by the agentic workflows (e.g., TAS, PTAS, SPEC, DRAFT).
- `⫻cmd/`: Commands for system control and ecosystem regulation (e.g., `⫻cmd/halt`).
- `⫻logic/`: Conditional logic regulating ecosystem flow and convergence.

## Operational Workflow
The ecosystem operates on a recursive loop, utilizing the `KickForge`, `KickFlow`, and `KickGuard` agent triad:
1.  **Extract**: Source data from the ecosystem.
2.  **Purify**: Process and refine raw data.
3.  **Orchestrate**: Plan structures and delegate tasks.
4.  **Consent Gate**: Enforce human-in-the-loop integrity (`⫻cmd/halt` if denied).
5.  **Review/Synthesize/Emit**: Finalize drafts and trigger the next recursive cycle.
